Detailed Description

A base class for all our singletons, which are mainly management behaviours.

It is set to be not destroyed on load and provides the usual static Get() and duplicate destruction. The subclass has to give its own type for the singleton to handle. You have to use MyAwake() instead of Awake() from MonoBehaviour. Also provides a conditional debug logging method and public switch.
