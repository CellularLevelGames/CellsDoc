Cells: Microbiome Warfare

Cells is a 2D top-down tactics game based on the Unity Engine. It will initially support mouse and keyboard controls and has a focus on multiplayer using Photon.

Current master version:

State: Initial draft with test scenes only!
Unity version to use: 2020.1.4f1
How to start: Check out the correct commit, install and assign the correct Unity version to the project in UnityHub and open the project. Nothing big to fire up yet, have a look at existing scenes.
