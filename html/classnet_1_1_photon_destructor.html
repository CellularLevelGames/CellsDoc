Detailed Description

Provides an RPC target for things to be destroyed by the master.

All Clients have to simply call the DestroyFromMaster() function to have an RPC raised automatically. The usage is debounced, so if this is called on the same object for multiple clients there should be no clash. If the RPC cannot land due to the object already having been destroyed we may get warnings though.
