◆ AddStatusEffect()

UnityAction AddStatusEffect ( StatusEffect  effect)

Adds the effect to the relevant status attribute.

The attribute itself is responsible for applying it and ending it after its duration. Returns an action that can be invoked for premature effect removal. Use either that or call the removal function with the same effect reference you gave here. Returns null if the effect didn't apply for the target.

Implemented in UnitStatus.
