Detailed Description

The GameManager controls the overall flow of the program and organizes the interaction of its parts.

It can be accessed via Get() by its Singleton pattern, after this object was initialized. It is not deleted after scene switch. It notably tracks the current State of the game, see also: GameStateMachine To not overcrowd this class, functionality is continuously relocated to external managers. See also: MatchManager PlayerManger NetManager
