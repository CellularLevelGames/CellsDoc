Detailed Description

This is to implement the target selection, as described here: Target Selection.

Intended usage:

  • Ensure this component is enabled, as also registering incoming opponents depends on it. Likewise, you can disable all activity by disabling the component.
  • use the inBattle bool to see if a battle is currently ongoing.
  • Scan for target immediately by using ScanForTargetsAndUpdate()
  • Use battleStartEvent and battleEndEvent to be notified of available targets. Note that the current target can switch at any time while a battle is ongoing.
