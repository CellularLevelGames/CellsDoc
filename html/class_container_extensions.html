inlinestatic

Returns a weighted random element from the enumerable.

You have to give a function able to determine the weight of each element. The weight will most likely be a member of the element, so the call may look like this when each element has a 'weight' member: listOfStuff.RandomElementByWeight(e => e.weight) Drawing elements without weights is of course as simple as determining a random index if that is all you need.
