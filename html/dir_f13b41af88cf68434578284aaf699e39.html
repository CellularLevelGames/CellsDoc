Detailed Description

Contains all C# scripts, sometimes together with closely linked resources.

Having resources next to the scripts should be an exception, which sometimes makes sense due to the close coupling of script and required other files, such as the state machine configuration for a state machine class. The only other location where scripts are held is the 'Test' folder, which is kept separate from the production code, together with all relevant resources for the test.
